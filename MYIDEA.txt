1. Wumpus World (Must Have)

This is literally a classic AI syllabus problem.

Game idea:
	•	Grid cave
	•	Player agent moves
	•	Sensors: breeze, smell
	•	Goal: find gold without dying

What students learn:
	•	Knowledge representation
	•	Logical inference

Features you can add:
	•	Step-by-step reasoning display
	•	AI agent solving it automatically

2. Maze Solver (Search Algorithms)

Students can visually see how search algorithms explore paths.

Game:
	•	Maze grid
	•	Start → Goal
	•	Algorithm finds path

Algorithms to visualize:
	•	BFS
	•	DFS
	•	Uniform Cost
	•	A*
	•	Greedy Best First

Show visually:
	•	nodes explored
	•	frontier
	•	final path

This teaches:
	•	Uninformed search
	•	Informed search

Very impressive visually.

3. 8 Puzzle Solver

Classic AI state-space problem.

Game:

1 2 3
4 5 6
7 _ 8

Goal:
Arrange tiles correctly.

Students can choose solver:
	•	BFS
	•	A*
	•	Greedy
	•	Manhattan heuristic

What they learn:
	•	State space search
	•	Heuristics
